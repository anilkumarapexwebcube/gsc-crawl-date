Data & privacy

          
Everything runs in your browser — no server stores your data. Search Console access tokens are
            kept only in this tab's sessionStorage and disappear when you close the tab. The OAuth Client
            ID,
            Apps Script URL and cached domain list live in localStorage on this device only. No client
            secret
            is ever used or stored.

          

        

      


      
      

        

          
How to use this tool

          
    
            
  1. Configure once. In the Setup tab, paste your Apps Script URL and your OAuth
              Web Client ID (with this page's origin added as an Authorized JavaScript origin), then Save.
            
    2. 
            
  2. Connect your Google accounts. Go to the Accounts tab, type each Google account you
              handle
              and click Connect. Sign in once — the token lasts for the browser session and refreshes silently.
    3. 
            
  3. Run reports. Paste one or more domains in the Run Audit tab, one per line. The preview
              under the textarea shows which account each domain uses and whether you're signed in. Pick a format and
              click Generate Report(s).
    4. 
            
  4. Find your reports. PPTX files download to your Downloads folder — about 10–20 seconds
              per
              domain for the James format.
    5.

Troubleshooting

          

            
              

                What you see
                What to do
              

            
            
              

                "Access blocked: no registered origin" / Error 401: invalid_client
                Your OAuth client doesn't list this page's address. In Setup, copy "This page's origin" and add it
                  under
                  Authorized JavaScript origins on your OAuth Web client, then re-enter the Client ID.
                
              

              

                "Not in master sheet" for a domain you expect to work
                Add it via the master Google Sheet, then click "Refresh domain list".
              

              

                "Session expired" for an account
                Accounts tab → Disconnect that account → Connect it again.
              

              

                Manual Action / Security / Sitemap slide is blank
                Upload a screenshot for it in the Screenshots tab — browsers can't capture the GSC UI automatically.
                
              

              

                Anything unexpected
                Clear the log, reproduce the issue, and screenshot the log panel in the Run Audit tab.
